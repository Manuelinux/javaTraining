SL275: Module4: Expressions and Control Flow

Exercise #1: Use Loops and Branching Statements

(Level 1)

Objective

In this exercise you will use a simple integer loop and branching statements to play a fictitious game of "foo bar baz".

Directions

Create an application that loops from 1 to 50 and prints out each value on a separate line, except print out "foo" for every multiple of 3, "bar" for every multiple of 5, and "baz" for every multiple of 7.

For example: 

1
2
3 foo
4
5 bar
6 foo
7 baz
8
9 foo
10 bar
11
12 foo
13
14 baz
15 foo bar
16
etc...

  1. Create a Java class called FooBarBaz with a single main method. In this method create a for loop statement with an iteration variable called number.

  2. Within the for loop, print out the number using the System.out.print method. This method, unlike its println cousin, does not print a newline character.

  3. Within the for loop, use an if statement to determine whether or not a "foo" should be printed for this number. If yes, then use the print method to print the string "foo". Do the same thing for the "bar" and "baz" cases. Hint -- The % operator calculates an integer remainder.

  4. Finally, also within the for loop, print a newline character using the System.out.println() method. This will close the line for this iteration.
