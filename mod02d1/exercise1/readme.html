Exercise #1: Use the Java API Documentation

(Level 1)

Objective

In this exercise you will explore the Java 2 SDK class API documentation using a Web browser. You will hunt for a specific method in the String class.

Directions

Using your Web browser perform the following actions:

  1. Open the "Java 2 Platform API Specification" page

  2. Select the java.text package in the package list in the upper left corner panel of the API frameset

  3. Select the NumberFormat class in the class list in the lower left panel

  4. Read about the class in the top part of the documentation window on the right panel of the frameset. Java does not support printing like C/C++ using printf controls to format numbers. Instead, these classes can be used to create printing control to and from strings. Review the format and parse methods.
